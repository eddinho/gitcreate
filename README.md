# gitcreate

The script creates a new github repo from a local repo

## Installation
```bash
git clone https://github.com/eddinho/gitcreate.git

sudo cp ~/dev/gitcreate/gitcreate /usr/local/bin/

sudo chmod +x /usr/local/bin/gitcreate
```
## Usage
```bash

gitcreate
```
